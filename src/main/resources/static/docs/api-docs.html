공지 API

HTTP 요청

요청 메세지

GET /api/v1/notice?type=bch&offset=0&max=10 HTTP/1.1
Accept: application/json
Host: kuring.herokuapp.com

요청 파라미터

Parameter Description

type

공지 카테고리 키워드

offset

가져올 공지의 시작 인덱스

max

가져올 공지 최대 개수

HTTP 응답

응답 메세지

HTTP/1.1 200 OK
Content-Type: application/json
Content-Length: 291

{
  "isSuccess" : true,
  "resultMsg" : "성공",
  "resultCode" : 200,
  "baseUrl" : "${notice.base-url}",
  "noticeList" : [ {
    "articleId" : "5cw2e1",
    "postedDate" : "20211016",
    "subject" : "[학사] 2021년 학사 관련 공지",
    "category" : "bachelor"
  } ]
}

응답 필드

Path Type Description

isSuccess

Boolean

성공 여부

resultMsg

String

결과 메세지

resultCode

Number

결과 코드

baseUrl

String

공지 확인할 수 있는 url의 공통 부분

noticeList[].articleId

String

공지 ID

noticeList[].postedDate

String

공지 게시일

noticeList[].subject

String

공지 제목

noticeList[].category

String

공지 카테고리명
